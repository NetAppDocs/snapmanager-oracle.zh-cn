= 验证 SnapManager 主机服务器状态
:icons: font
:imagesdir: ../media/


[role="lead"]
要执行命令或启动 SnapManager 操作，服务器必须正在运行。在执行任何操作之前，您必须验证服务器的状态。

. 在服务窗口中，选择适用于 Oracle 的 SnapManager 3.3 。
. 在状态列中查看状态。
