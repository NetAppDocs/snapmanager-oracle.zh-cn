= 升级后任务
:icons: font
:imagesdir: ../media/


[role="lead"]
升级到更高版本的 SnapManager 后，您必须更新现有存储库。您可能还需要修改分配给现有备份的备份保留类。


NOTE: 升级到 SnapManager 3.3 或更高版本后，如果要使用数据库（ DB ）身份验证作为唯一的身份验证方法，则需要将 sqlnet.authentication_services 设置为 none 。RAC 数据库不支持此功能。
