= smo profile show 命令
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以运行 profile show 命令来显示有关配置文件的信息。



== 语法

[listing]
----

        smo profile show
-profile profile_name
\[-quiet \| -verbose\]
----


== Parameters

* -profile profile_name*
+
指定配置文件的名称。此名称最长为 30 个字符，并且在主机中必须是唯一的。

* * —静默 *
+
在控制台中仅显示错误消息。默认情况下会显示错误和警告消息。

* * —详细 *
+
在控制台中显示错误，警告和信息性消息。
