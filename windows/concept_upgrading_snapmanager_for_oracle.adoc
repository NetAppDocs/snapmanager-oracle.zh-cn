= 升级 SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以从任何早期版本升级到最新版本的 SnapManager for Oracle 。您可以同时升级所有 SnapManager 主机，也可以执行滚动升级，以便按主机交错升级主机。
