= smo credential clear 命令
:icons: font
:imagesdir: ../media/


[role="lead"]
此命令将清除所有受保护资源的用户凭据缓存。



== 语法

[listing]
----

        smo credential clear
\[-quiet \| -verbose\]
----


== Parameters

* * —静默 *
+
在控制台上仅显示错误消息。默认情况下会显示错误和警告消息。

* * —详细 *
+
在控制台上显示错误，警告和信息性消息。





== 命令示例

此示例将清除运行命令的用户的所有凭据。

[listing]
----
smo credential clear -verbose
----
[listing]
----
SMO-20024 [INFO ]: Cleared credentials for user "user1".
----
