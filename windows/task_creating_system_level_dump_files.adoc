= 创建系统级转储文件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用 smo system dump 命令获取有关 SnapManager 主机和环境的日志信息。您可以指定不同的日志级别来收集有关特定操作，配置文件或主机和环境的信息。

. 输入以下命令： smo system dump
+
生成的转储

+
[listing]
----
Path:\<user-home>\Application Data\NetApp\smo\3.3.0\smo_dump_server_host.jar
----
