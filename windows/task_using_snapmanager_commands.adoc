= 使用 SnapManager 命令
:icons: font
:imagesdir: ../media/


[role="lead"]
启动 SnapManager 主机服务器后，您可以通过在主机上的提示符处输入命令来使用 SnapManager 。

. 要执行操作，请执行以下操作：
+
** 对于 Windows 主机，请转到 * 开始 * > * 所有程序 * > * NetApp* > * SnapManager for Oracle* > * 启动 SMO 命令行界面（命令行界面） *
