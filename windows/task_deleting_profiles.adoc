= 正在删除配置文件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以随时删除配置文件，只要该配置文件不包含成功或不完整的备份即可。您可以删除包含已释放或已删除备份的配置文件。

. 要删除配置文件，请输入以下命令： smo profile delete -profile profile_name
