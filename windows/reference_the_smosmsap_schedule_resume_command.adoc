= smo schedule resume 命令
:icons: font
:imagesdir: ../media/


[role="lead"]
此命令将恢复暂停的备份计划。



== 语法

[listing]
----

        smo schedule resume
-profile profile_name
-schedule-name schedule_name
\[-quiet \| -verbose\]
----


== Parameters

* -profile profile_name*
+
指定与要恢复暂停的备份计划的数据库相关的配置文件的名称。配置文件包含数据库的标识符和其他数据库信息。

* -schedule-name schedule_name*
+
指定创建备份计划时提供的计划名称。
