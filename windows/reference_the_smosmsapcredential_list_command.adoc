= smo credential list 命令
:icons: font
:imagesdir: ../media/


[role="lead"]
此命令将列出用户的所有凭据。



== 语法

[listing]
----

        smo credential list
\[-quiet \| -verbose\]
----


== Parameters

* * —静默 *
+
在控制台上仅显示错误消息。默认情况下会显示错误和警告消息。

* * —详细 *
+
在控制台上显示错误，警告和信息性消息。





== 命令示例

此示例显示了运行命令的用户的所有凭据。

[listing]
----
smo credential list
----
