= 删除存储库的用户凭据
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以删除凭据，以便用户无法再访问特定存储库。使用此命令，您可以仅删除一个资源的凭据，而不是清除所有资源的用户凭据。

. 要删除用户的存储库凭据，请输入以下命令： "mo credential delete -repository-dbnamerepo_service_name-host-login -usernamerepo_username-portrepo_port"
