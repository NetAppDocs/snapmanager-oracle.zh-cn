= smo clone split result 命令
:icons: font
:imagesdir: ../media/




== 语法

使用此命令可以查看克隆拆分过程的结果。

[listing]
----

        smo clone split-result
-profile profile
\[-host hostname\]
\[-label split-label \| -id split-id\]
\[-quiet \| -verbose\]
----


== Parameters

* * 配置文件配置文件 *
+
指定克隆的配置文件名称。

* * —主机主机主机名 *
+
指定克隆所在的主机名。

* * —标签拆分标签 *
+
指定克隆拆分启动进程生成的标签名称。

* * — id 拆分 ID *
+
指定克隆拆分启动进程生成的唯一 ID 。

* * —静默 *
+
在控制台上仅显示错误消息。默认情况下会显示错误和警告消息。

* * —详细 *
+
在控制台上显示错误，警告和信息性消息。
