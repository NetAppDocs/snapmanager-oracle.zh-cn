= 查看克隆拆分过程的结果
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看已启动的克隆拆分过程的结果。

. 要查看克隆拆分过程的结果，请输入以下命令： `smo clone split -result -profileprofile "-hosthostname] "-labelsplit -label | -idsplit -id] "-quiet | -verbos]`


[listing]
----
smo clone split-result -profile p1 -id 8abc01ec0e78f3e2010e78f3fdd00001
----
