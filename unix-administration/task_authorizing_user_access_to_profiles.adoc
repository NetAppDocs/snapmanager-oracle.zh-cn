= 授权对配置文件的访问
:icons: font
:imagesdir: ../media/


[role="lead"]
除了基于角色的访问控制（ Role-Based Access Control ， RBAC ）之外，您还可以通过 SnapManager 为配置文件设置密码，以防止未经授权的访问。

. 输入以下命令： "smo credential set -profile -name profile_name [-password password]"
