= 重新启动 SnapManager UNIX 主机服务器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用命令行界面在 UNIX 主机上重新启动服务器。

. 要重新启动服务器，请输入以下命令： `smo_server restart`
