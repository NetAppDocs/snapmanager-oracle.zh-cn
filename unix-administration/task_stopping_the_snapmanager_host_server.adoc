= 正在停止 SnapManager 主机服务器
:icons: font
:imagesdir: ../media/


[role="lead"]
使用完 SnapManager 后，您可能需要停止服务器。

. 要停止服务器，请以 root 用户身份输入以下命令： `smo_server stop`
