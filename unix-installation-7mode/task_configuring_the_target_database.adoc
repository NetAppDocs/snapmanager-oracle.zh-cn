:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
目标数据库是 Oracle 数据库，可以配置为独立， Real Application Clusters （ RAC ）， Automatic Storage Management （ ASM ）或任何其他受支持的组合。

. 请参见 _TR-3633_ 来配置目标数据库。
