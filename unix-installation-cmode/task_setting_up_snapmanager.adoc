= 设置 SnapManager
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用用户界面（ UI ）或命令行界面（ CLI ）启动 SnapManager 并访问它。访问 SnapManager 后，您必须先创建 SnapManager 存储库，然后再执行任何 SnapManager 操作。
